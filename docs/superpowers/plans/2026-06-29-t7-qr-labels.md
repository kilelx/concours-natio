# T7 — QR Labels Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `/labels` page that renders a printable grid of QR sticker labels for all 12 FACOM tools.

**Architecture:** Client component fetches tools once from `/api/tools` on mount and renders a grid of `QrLabel` components. Labels are static so no SSE needed. Print styles force white background and hide navigation controls. The page is `"use client"` to allow the inline `window.print()` button — avoids server/client split complexity.

**Tech Stack:** Next.js 16 App Router, React 19, `qrcode.react`, Tailwind v4, TypeScript.

## Global Constraints

- Next.js 16.2.9 — App Router, server components by default
- `"use client"` required for any component using browser APIs or React hooks
- Props passed from server to client components must be serializable (no Maps, no functions)
- Tailwind v4 syntax — use `print:hidden` for print-hidden elements
- Dark theme via `bg-gray-950 text-white` — must be overridden to white for print

// components/qr-label.tsx
"use client"

import { QRCodeSVG } from "qrcode.react"
import type { Tool } from "@/lib/types"

export function QrLabel({ tool }: { tool: Tool }) {
  return (
    <div className="bg-white text-black border border-dashed border-gray-400 rounded p-3 flex flex-col items-center gap-2 w-full aspect-[3/2]">
      <QRCodeSVG value={tool.id} size={96} level="M" />

        <div className="font-mono text-xs font-bold text-gray-700">{tool.id}</div>
        <div className="text-xs font-medium text-gray-900 leading-tight mt-0.5 line-clamp-2">{tool.name}</div>
        <div className="text-xs text-gray-500 mt-0.5">{tool.reference}</div>
      </div>
    </div>
  )
}

### Task 3: Labels page

**Files:**
- Create: `app/labels/page.tsx`

**Interfaces:**
- Consumes: `GET /api/tools` → `{ tools: Tool[] }` — one-shot fetch on mount
- Consumes: `QrLabel` from `@/components/qr-label`
- Produces: route `/labels` — printable grid of all tools

- [ ] **Step 1: Create the page**

```tsx
// app/labels/page.tsx
"use client"

import { useEffect, useState } from "react"
import { QrLabel } from "@/components/qr-label"
import type { Tool } from "@/lib/types"

export default function LabelsPage() {
  const [tools, setTools] = useState<Tool[]>([])

  useEffect(() => {
    fetch("/api/tools")
      .then((r) => r.json())
      .then((data) => setTools(data.tools))
  }, [])

  return (
    <>
      <style>{`
        @media print {
          body { background: white !important; }
          .no-print { display: none !important; }
          .label-grid { gap: 0 !important; padding: 0 !important; }
        }
      `}</style>

      <div className="min-h-screen bg-gray-950 text-white">
        {/* Header — hidden on print */}
        <div className="no-print flex items-center justify-between p-4 border-b border-gray-800">
          <div className="flex items-center gap-3">
            <div className="w-8 h-8 bg-red-600 rounded flex items-center justify-center font-bold text-sm">F</div>
            <div>
              <h1 className="text-xl font-bold tracking-tight">Étiquettes QR</h1>
              <p className="text-xs text-gray-400">{tools.length} outils · Prêt à imprimer</p>
            </div>
          </div>
          <div className="flex items-center gap-3">
            <a href="/" className="text-xs text-gray-400 hover:text-white transition-colors">
              ← Dashboard
            </a>
            <button
              onClick={() => window.print()}
              className="bg-red-600 hover:bg-red-700 text-white text-sm font-medium px-4 py-2 rounded transition-colors"
            >
              Imprimer
            </button>
          </div>
        </div>

        {/* Label grid */}
        <div className="label-grid p-6 grid grid-cols-4 gap-4">
          {tools.map((tool) => (
            <QrLabel key={tool.id} tool={tool} />
          ))}
        </div>
      </div>
    </>
  )
}
```
